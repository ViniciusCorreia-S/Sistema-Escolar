
# 📚 Sistema de Gestão Escolar

Sistema de Gestão Escolar desenvolvido em C# com o objetivo de praticar conceitos de Programação Orientada a Objetos (POO) e organização de sistemas em camadas.

O projeto simula o funcionamento básico de um sistema escolar, permitindo gerenciar alunos, professores e turmas, além de realizar operações de cadastro e gerenciamento dessas entidades.

Sistemas de gestão escolar são utilizados para organizar informações acadêmicas, controlar turmas, alunos e registros escolares, facilitando a administração das instituições de ensino.


## 🚀 Funcionalidades

- 👨‍🎓 Cadastro de alunos
- 👨‍🏫 Cadastro de professores
- 🏫 Criação e gerenciamento de turmas
- 📋 Listagem de registros cadastrados
- ➕ Adição de alunos às turmas
- ❌ Remoção de alunos, professores e turmas
- 📊 Organização das informações escolares


## 🛠️ Tecnologias utilizadas

- C#
- .NET
- Programação Orientada a Objetos (POO)
- Console Application
- Biblioteca Spectre Console
